# Trading Analytics Portal

Upload PDF trading reports and get instant analytics.

## Deploy to Vercel

1. Push this folder to GitHub
2. Go to vercel.com
3. Import your repo
4. Deploy

## Features

- PDF upload and parsing
- Client performance analytics
- Profit/Loss tracking
- CSV export
- Responsive design

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000
